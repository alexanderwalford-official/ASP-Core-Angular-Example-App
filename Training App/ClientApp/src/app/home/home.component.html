Important Information

Why did you make this?

For my GitHub portfolio, I have created a simple Angular ASP .NET Core application that can perform basic operations with the API backend to showcase my programming capabilities.


What can this app do?

Good question! For starters, it can only do some very basic things to illustrate the differences between the backend and frontend libraries. This may include, but is not limited to: - Simplistic Angular webpages, including components from popular libraries. - Basic API requests with C# ASP Core controllers. - Illustrates the use of front end components and UI libraries.
